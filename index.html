Manual Sistem

Visi: Mengintegrasikan seluruh data administrasi ke dalam satu gerbang akses terenkripsi untuk mewujudkan transformasi kesehatan digital.

Kerja Sistem:
  • 1. Frontend: Interface berbasis Tailwind CSS yang dioptimalkan untuk akses responsif via Smartphone, Tablet, maupun Desktop.
  • 2. Authentication: Protokol keamanan pintu berlapis menggunakan verifikasi token enkripsi SECRET_TOKEN.
  • 3. Data Layer: Integrasi Google Apps Script (GAS) sebagai *bridge* (jembatan) data real-time antara Web Interface dan Cloud Database.
  • 4. Security: Proteksi sesi login berbasis enkripsi lokal untuk menjaga kerahasiaan data selama masa aktif peramban (browser).

Log Keamanan Sesi
